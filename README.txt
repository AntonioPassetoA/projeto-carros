
# Projeto Carros - Fullstack (React + Node + MySQL)

## Como rodar:

1. Configure o MySQL (usuário, senha e banco carrosdb):
   - CREATE DATABASE carrosdb;
   - Depois ajuste DB_USER e DB_PASS no arquivo backend/.env

2. Rode o backend:
   cd backend
   npm install
   npm start

   (O backend cria as tabelas e usuário admin/123456 na primeira execução)

3. Rode o frontend:
   cd frontend
   npm install
   npm start

4. Acesse http://localhost:3000 no navegador

Senha: 123456
